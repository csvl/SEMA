76 def step_state(self, simgr, state, **kwargs): # pylint:disable=no-self-use 

77 """ 

78 Determine the categorization of state successors into stashes. The result should be a dict mapping stash names 

79 to the list of successor states that fall into that stash, or None as a stash name to use the original stash 

80 name. 

81 

82 If you would like to directly work with a `SimSuccessors` object, you can obtain it with 

83 ``simgr.successors(state, **kwargs)``. This is not recommended, as it denies other hooks the opportunity to 

84 look at the successors. Therefore, the usual technique is to call ``simgr.step_state(state, **kwargs)`` and 

85 then mutate the returned dict before returning it yourself. 

86 

87 ..note:: This takes precedence over the `filter` hook - `filter` is only applied to states returned from here 

88 in the None stash. 

89 

90 :param angr.SimulationManager simgr: 

91 :param angr.SimState state: 

92 """ 

93 return simgr.step_state(state, **kwargs)
